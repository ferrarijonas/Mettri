Ativar GitHub Pages e rodar o deploy

Abra os links abaixo (você precisa estar logado no GitHub).

1. Ativar GitHub Pages
Abrir Settings → Pages
  1. Em Source, escolha Deploy from a branch.
  2. Branch: gh-pages. Folder: / (root).
  3. Clique em Save.
2. Rodar o deploy (criar branch gh-pages)
Abrir workflow Deploy modules
