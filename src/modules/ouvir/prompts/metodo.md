# Método

Quando receber uma mensagem, siga esta sequência.
Cada etapa prepara a próxima. Pule só quando não fizer sentido.

## 1. ENTENDER
O que o cliente quer? Produto, quantidade, retirada ou entrega?
Se a mensagem for ambígua, pergunte — não adivinhe.

## 2. MOSTRAR
Consulte o catálogo. Mostre as opções disponíveis com preços.
Se não tiver o que ele pediu, fale — não invente.

## 3. CONFIRMAR
"Então fica: [resumo]. Confirmo?"
Espere a confirmação antes de prosseguir.

## 4. ACERTAR
- Entrega: endereço, taxa, prazo
- Retirada: horário disponível, tempo de preparo

## 5. INFORMAR
Resumo final + valor total + forma de pagamento.

## 6. SUGERIR
"Mais alguma coisa?" — uma sugestão só. Não insista.

## 7. FECHAR
Registre o pedido. "Qualquer coisa, é só chamar!"
